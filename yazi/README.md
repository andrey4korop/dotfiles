# Yazi File Manager

This directory contains the configuration for [Yazi](https://github.com/sxyazi/yazi), a terminal file manager written in Rust.

## Files

*   `.config/yazi/`: The main directory for Yazi's configuration.
    *   `init.lua`: A Lua script that is run on startup, allowing for advanced custom initialization.
    *   `keymap.toml`: Defines custom key bindings for navigating and operating Yazi.
    *   `package.toml`: A file that might be related to Yazi's plugin or package management system.
    *   `yazi.toml`: The primary configuration file for Yazi, containing settings for the UI, behavior, and other options.
    *   `plugins/`: A directory to store Yazi plugins.
