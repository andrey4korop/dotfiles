# MangoHud

This directory contains the configuration for [MangoHud](https://github.com/flightlessmango/MangoHud), a Vulkan and OpenGL overlay for monitoring FPS, temperatures, CPU/GPU load and more.

## Files

*   `.config/MangoHud/MangoHud.conf`: The main configuration file for MangoHud. It allows you to customize which performance metrics are displayed, their appearance, and other settings for the overlay.
