# Midnight Commander (mc)

This directory contains configuration files for [Midnight Commander](https://midnight-commander.org/), a visual file manager.

## Files

*   `.config/mc/`: This directory holds the user-specific configurations for `mc`.
    *   `filehighlight.ini`: Defines file highlighting rules based on file types and permissions.
    *   `hotlist`: Stores the user's "hotlist" of frequently visited directories.
    *   `ini`: The main configuration file for Midnight Commander, storing settings for the panels, editor, viewer, and overall behavior.
    *   `mc.ext.ini`: An extension file for defining actions based on file extensions (e.g., what happens when you press Enter on a `.zip` file).
    *   `menu`: User-defined menu items for the `mc` interface.
    *   `panels.ini`: Stores settings related to the file panels, such as display mode, sorting, and other panel-specific options.
    *   `mcedit/`: This directory could contain settings or snippets for the built-in editor, `mcedit`.
